摘要：本次课程介绍了淋巴系统的组成与功能，主要包括淋巴管道、淋巴组织和淋巴器官三部分。淋巴液通过毛细淋巴管吸收组织液形成，并在淋巴管道中单向流动，经淋巴结过滤后汇入静脉，参与血液循环。淋巴系统具有回收蛋白质、运输营养物质、调节液体平衡及清除异物等作用。淋巴管道包括毛细淋巴管、淋巴管、淋巴干和淋巴导管，最终汇合成右淋巴导管和胸导管。淋巴组织以网状结构为基础，富含淋巴细胞和巨噬细胞，分为弥散淋巴组织和淋巴小结，体现特异性、转化性和记忆性。主要淋巴器官包括淋巴结、扁桃体、脾和胸腺，分别承担滤过淋巴液、免疫应答、造血与储血以及T细胞发育等功能。

# 一、淋巴系统的组成与总体功能
淋巴系统由淋巴管道、淋巴组织和淋巴器官组成，淋巴液沿淋巴管道流动并最终汇入静脉，与心血管系统共同构成脉管系统，参与物质运输、内分泌调节和机体防御

# 二、淋巴管道的结构与分类
- 淋巴管道为单向回流系统，起始于组织间隙的毛细淋巴管（盲端结构），依次汇合为淋巴管、淋巴干和淋巴导管
- 毛细淋巴管通透性高，可吸收蛋白质、细菌和癌细胞等大分子物质

# 三、淋巴管的分布与特点
- 淋巴管分为浅淋巴管和深淋巴管，二者间有广泛吻合
- 因淋巴回流速度仅为静脉的1/10，故其数量和瓣膜数目远多于静脉，以保障正常回流

# 四、淋巴干与淋巴导管
- 全身共有9条淋巴干:- 左右颈干、左右锁骨下干、左右支气管纵隔干、左右腰干及肠干
- 这些淋巴干汇集成右淋巴导管和胸导
- 右淋巴导管收纳约全身1/4区域的淋巴，胸导管为最大淋巴管，起自乳糜池，收纳约3/4区域的淋巴

# 五、淋巴组织的结构与特征
1. 淋巴组织又称免疫组织，以网状组织为支架，内含大量淋巴细胞、巨噬细胞和浆细胞
2. 分为弥散淋巴组织和淋巴小结两种形态
3. 淋巴细胞具有特异性、转化性和记忆性，维持免疫动态平衡
# 六、淋巴器官的种类与功能
- 淋巴器官包括淋巴结、扁桃体、脾和胸腺
- 淋巴结遍布全身，约800多个，具有滤过淋巴液、产生淋巴细胞和参与免疫反应的功能
![[07.淋巴结模式图.jpg|725]]
# 七、扁桃体的位置与免疫作用
扁桃体位于消化道与呼吸道交汇处，包括腭扁桃体、咽扁桃体和舌扁桃体，黏膜内富含淋巴组织，是抗原接触和局部免疫应答的重要部位

# 八、脾的结构与生理功能
脾位于左季肋区，胃左侧与膈之间，胚胎期具造血功能，出生后生成淋巴细胞和单核细胞，可吞噬衰老红细胞、血小板及异物，并能储存血液，在失血或运动时通过平滑肌收缩释放储备血
![[07.脾的位置和形态.jpg|549]]
# 九、胸腺的位置与免疫内分泌功能
- 胸腺位于胸骨柄后方，为中枢淋巴器官，促进T细胞发育成熟，兼具内分泌功能
- 规律的有氧运动有助于增强免疫力，降低上呼吸道感染风险
