摘要：本次课程主要介绍了体循环静脉系统的结构与功能，包括静脉的分布规律、体循环静脉的组成及其血液回流路径。体循环静脉负责将全身各部位富含二氧化碳和代谢产物的静脉血经上腔静脉、下腔静脉等返回右心房。静脉系统分为深静脉和浅静脉，具有丰富的吻合支，部分结构如静脉瓣、硬脑膜窦等具有特殊功能。课程重点讲解了上腔静脉系、下腔静脉系及肝门静脉系的主要属支、走行和引流范围，并强调了肝门静脉在物质运输和侧支循环中的重要作用。

---

# 一、体循环静脉概述
- 体循环静脉将毛细血管处完成物质交换后的静脉血（富含二氧化碳和代谢废物）经各级静脉汇入上腔静脉、下腔静脉，最终回流至右心房
- 血液中二氧化碳通过肺循环排出，代谢废物经尿液等途径排出体外

# 二、静脉分布规律
![[05.全是静脉主分支.jpg|400]]
1. 静脉始于毛细血管，逐级汇合成小、中、大静脉干
2. 数量多、管腔大、管壁薄、弹性小、血流速度慢
3. 多数静脉有静脉瓣防止血液倒流
4. 分为深静脉与浅静脉两类，二者之间存在广泛交通支

# 三、深静脉与浅静脉特点
1. 深静脉位于深筋膜深面，常与同名动脉伴行，一般中等动脉伴随两条静脉
2. 浅静脉位于皮下浅筋膜内，又称皮下静脉，不与动脉伴行，最终注入深静脉
3. 静脉间存在丰富吻合，包括浅-浅、深-深、浅-深静脉之间的交通

# 四、特殊静脉结构
硬脑膜窦为颅内特殊静脉结构，无平滑肌和瓣膜，对颅脑静脉血回流具有重要作用

# 五、体循环静脉主要组成
主要包括上腔静脉系、下腔静脉系和心静脉系
这本书重点介绍上腔静脉系和下腔静脉系

# 六、上腔静脉系
1. 收集头颈、上肢、胸壁及部分胸腔脏器的静脉血
2. 主干为上腔静脉，由左右头臂静脉在右侧第一胸肋结合处后方汇合而成，沿升主动脉右侧下行注入右心房
3. 头臂静脉由同侧颈内静脉和锁骨下静脉在胸锁关节后方汇合形成，汇合处称静脉角，为淋巴导管注入部位

# 七、头颈部静脉
![[05.头部及颈部静脉.jpg|504]]
- 深静脉主要为颈内静脉，起自颅底颈静脉孔，接受颅内、甲状腺及头面部静脉血
- 浅静脉为颈外静脉，起自下颌角，跨越胸锁乳突肌表面注入锁骨下静脉
![[05.面部静脉.jpg|497]]
# 八、上肢静脉
![[05.上肢钱静脉.jpg|415]]
1. 分为深静脉和浅静脉，均最终汇入腋静脉
2. 深静脉与同名动脉伴行，多为两条，汇合成肱静脉后形成腋静脉
3. 腋静脉跨第一肋外缘移行为锁骨下静脉
4. 浅静脉包括头静脉、贵要静脉和肘正中静脉
5. 头静脉起自手背静脉网桡侧，注入腋静脉或锁骨下静脉
6. 贵要静脉起自手背静脉网尺侧，注入肱静脉或腋静脉
7. 肘正中静脉连接头静脉与贵要静脉

# 九、胸部静脉
![[05.奇静脉及其分支.jpg|535]]
主干为肋间后静脉，汇入上腔静脉,重要属支包括半奇静脉、副半奇静脉及椎静脉丛等
# 十、下腔静脉系
- 收集膈以下下半身静脉血，主干为下腔静脉，是人体最粗大的静脉
- 由左右髂总静脉在第四、五腰椎右前方汇合而成，沿腹主动脉右侧上行，经腔静脉孔穿膈进入胸腔，注入右心房

# 十一、下腔静脉属支
![[05.肝门静脉及其分支.png|447]]
分为壁支和脏支
1. 壁支包括一对膈下静脉和四对腰静脉
2. 脏支分为成对与不成对两类
3. 成对脏支包括睾丸静脉（男性）、卵巢静脉（女性）、肾静脉、肾上腺静脉和肝静脉
4. 不成对脏支先汇入肝门静脉，再经肝静脉回流至下腔静脉

# 十二、髂总静脉与下肢静脉
![[05.下腔静脉.png|512]]
髂总静脉由髂内静脉和髂外静脉在骶髂关节前方汇合而成
- 髂内静脉为盆部主干，属支包括臀上、臀下静脉、闭孔静脉、骶外侧静脉（壁支）及直肠下静脉、阴部内静脉、子宫静脉（脏支）
- 髂外静脉为股静脉延续，收集下肢和腹前壁下部静脉血，主要属支为腹壁下静脉

# 十三、下肢浅静脉
![[05.下肢静脉.jpg|655]]
足背静脉弓由足背浅静脉吻合形成
- 大隐静脉起自足背静脉弓内侧，经内踝前方沿下肢内侧上行注入股静脉，收集足、小腿内侧、大腿前内侧、大腿外侧、脐下腹前壁及外阴部浅层静脉血
- 小隐静脉起自足背静脉弓外侧，经外踝后方沿小腿后面上行，于腘窝注入腘静脉，沿途收集足外侧及小腿后部浅静脉

# 十四、肝门静脉系
![[05.上下腔静脉的吻合和门静脉及其分支.png|428]]
由肝门静脉及其属支组成，收集食管下段、胃、小肠、大肠上部、胆囊及腹腔不成对器官的静脉血
主要属支包括肠系膜上静脉、肠系膜下静脉、脾静脉、胃左静脉、胃右静脉、胆囊静脉和附脐静脉；肝门静脉将消化道吸收的物质运送至肝进行合成分解、解毒和储存，是肝的功能性血管。

# 十五、肝门静脉的侧支循环
肝门静脉与上、下腔静脉系统之间存在丰富吻合,当肝门静脉因病变回流受阻时，可通过这些吻合形成侧支循环，维持血液回流
