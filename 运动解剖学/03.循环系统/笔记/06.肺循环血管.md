摘要：本次课程介绍了肺循环血管的结构与功能，重点讲解了血液从右心室经肺动脉、肺毛细血管进行气体交换后，再通过肺静脉回流至左心房的过程。肺循环中，静脉血在肺泡处转化为含氧丰富的动脉血，实现气体交换的关键结构是气血屏障。课程还详细阐述了肺动脉、肺静脉的走行与分支特点，以及肺泡的组织结构和表面活性物质的作用。

---

# 一、肺循环路径与功能
血液由右心室射出，经肺动脉及其分支到达肺泡毛细血管，在此进行气体交换，将静脉血转化为动脉血，再经肺静脉进入左心房，该过程称为肺循环或小循环

# 二、肺动脉的结构与分布
肺动脉起于右心室，属弹性动脉，斜行向上至主动脉弓下方分为左右肺动脉，经肺门入肺后随支气管分支形成毛细血管网，分布于肺泡壁周围

# 三、肺静脉的汇流路径
1. 肺静脉起自肺内毛细血管，逐级汇合成左上、左下、右上、右下肺静脉
2. 左上肺静脉收集左肺上叶血液，左下肺静脉收集左肺下叶血液，右上肺静脉收集右肺上中叶血液，右下肺静脉收集右肺下叶血液，最终均注入左心房

# 四、肺泡的结构特征
- 肺泡为半球形囊泡，成人约有3~4亿个，平均直径0.2毫米，总表面积约100平方米
- 肺泡壁由单层扁平上皮及基膜构成，含Ⅰ型肺泡细胞（扁平细胞）和Ⅱ型肺泡细胞（立方形分泌细胞），二者间形成紧密连接

# 五、表面活性物质与肺泡稳定性
Ⅱ型肺泡细胞分泌表面活性物质，覆盖于肺泡内表面液体层，可降低肺表面张力，维持肺泡壁的稳定性，防止肺塌陷

# 六、肺泡隔的组成与功能
- 肺泡隔位于相邻肺泡之间，内含丰富毛细血管、弹性纤维、胶原纤维和巨噬细胞
- 弹性纤维赋予肺泡良好的弹性和扩张性，巨噬细胞具有吞噬尘埃、细菌、异物和红细胞的能力

# 七、气血屏障的结构与气体交换机制
- 气血屏障是肺泡与血液间气体交换所通过的结构，包括肺泡表面液体层、Ⅰ型肺泡细胞及其基膜、肺泡隔结缔组织、毛细血管基膜及连续性内皮
- 氧气必须依次通过:血管内皮->血管内皮基膜->肺泡上皮基膜->肺泡上皮
- 氧气由肺泡向血液扩散，二氧化碳由血液向肺泡扩散，实现气体交换

# 八、气体交换的动力与结果
- 由于肺泡内氧分压高于血液、血液中二氧化碳分压高于肺泡，促使氧气进入血液、二氧化碳进入肺泡
- 交换后血液氧含量升高、二氧化碳含量降低，成为富含氧的动脉血，经体循环输送至全身
